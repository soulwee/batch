一、说明
	1.适用环境：屈臣氏
    2.发布时间：2018年11月07日
	3. www/data.txt为该项目的数据源文件, 数据源格式为 用户名|项目式课程id
	4.日志文件为根目录的log.txt（自动生成）

二、功能(修复内容)
	1.该批处理文件的功能：根据用户名和项目式课程id，将项目式课程关联的实操题审批的状态修改为通过
	（客户说法为数据源中的项目式课程在地图节点中单独使用，并且课程都关联实操题审批）
	
三、执行步骤
	1.配置www/jdbc.properties的数据源，此处的用的驱动包为ojdbc6.jar
	2.执行database/db_changes_oracle.sql脚本，创建中间表和索引
	3.执行 test.sh/bat
	4.如果启动错误，可以查看是否为【运行异常文档记录】中的问题
	5.log.txt 出现【update end】 为执行完成
